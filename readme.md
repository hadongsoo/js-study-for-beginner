## 단계별 환경

    로컬에 환경 세팅해보고 진행하는 것도 좋으나, 
    빠른 코드 작성 > 결과 확인이 1순위

1. 입문 : 크롬 devtool > console 탭 열고 바로 쳐보고 결과 확인
2. 기초 : codepen 또는 codesandbox 에 코드 작성후 결과 확인
3. 숙달 : 사이드 프로젝트 계획해서 github 계정에 연동
